# Demo Videos

Add compressed, captioned portfolio demos using these names:

- `classification-demo.mp4`
- `counting-demo.mp4`
- `segmentation-demo.mp4`
- `object-detection-demo.mp4`

Large video files are ignored by Git, so the named MP4s are intentionally not
committed as invalid empty files. For a public portfolio, host real demos in a
GitHub Release, object storage, or a video platform and link them from the main
README. Never include customer images or sensitive training data.
